# 3D AI Weather Chatbot

Dark 3D weather dashboard with current weather, hourly chart, 5-day forecast cards, weather details, alerts, and AI chatbot.

## Run locally

```bash
pip install -r requirements.txt
streamlit run weatherchatbot.py
```

## API keys

Create a `.env` file in this folder:

```env
OPENWEATHER_API_KEY=your_openweather_key
HF_TOKEN=your_huggingface_token
```

Or paste keys in the sidebar after running the app.
